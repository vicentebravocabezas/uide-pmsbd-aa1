# Aprendizaje Autónomo 1 — Granito Coffee Shop

Sitio web del emprendimiento digital **Granito Coffee Shop**.

## Descargar

- [Linux](https://github.com/vicentebravocabezas/uide-pmsbd-aa1/releases/download/latest/aa1-linux.zip)
- [Windows](https://github.com/vicentebravocabezas/uide-pmsbd-aa1/releases/download/latest/aa1-windows.zip)

El archivo descargado será un zip con el servidor, el directorio `frontend` y la base de datos de sqlite `db.sqlite3`

Para ejecutarlo es necesario descomprimir los archivos y abrir desde la terminal o el **CMD** el archivo `aa1-windows.exe` y se abrira el servidor en el [localhost al puerto **3000**](http://localhost:3000). Aquí se podrá utilizar el sitio sin necesidad de ningún programa adicional.

## Arquitectura

- Backend en Go
- Base de datos en SQLITE
- jQuery 4 importado como módulo en archivos de JavaScript
